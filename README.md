
# My First Python Calculator

Hello! This is my project for Assignment 6. I have built a GUI (Graphical User Interface) Calculator using Python and the Tkinter library.

## What I Learned
Through this project, I learned:
- How to create windows and buttons using **Tkinter**.
- How to use **Global Variables** to store numbers and operators while the program is running.
- How to write **Manual Arithmetic Logic** instead of using shortcuts like `eval()`.
- How to handle user input and display results on a screen.

## Project Features
- **Manual Calculations:** I wrote specific logic for addition, subtraction, multiplication, and division.
- **Clear Button:** Resets the calculator so you can start over.
- **Responsive Layout:** Used the `.grid()` system to align all buttons perfectly.

## Files in this Folder
- `calculator.py`: The main code for the calculator.
- `README.md`: This file, explaining the project.

## How to Run
1. Open your terminal or command prompt.
2. Go to the folder where you saved these files.
3. Type `python calculator.py` and press Enter.
