# Calculator Project by Sakshi

This is my assignment for a Tkinter calculator. 

How it works:
- I used buttons to take numbers from the user.
- I used global variables like 'first_val' to store the first number when you click +, -, *, or /.
- I didn't use the eval function. I wrote 'if' statements to do the math myself.
